# `mev_out/` (generated results)

This folder is the default output location for the repo’s analysis scripts
(e.g., `scripts/mev_collect.py`, `scripts/section3_empirical*.py`).

Contents are **generated** (CSVs, HTML plots, PDFs/PNGs) and are ignored by git by
default. Curated snapshots for GitHub Pages live under `docs/assets/`.
